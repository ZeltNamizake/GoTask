## Changelog

## [v1.0.0] - Initial Release

- Task Management:
  - Add task
  - Mark as done / undone
  - Delete task
  - Edit task (title, start time, done time)
  - Show tasks (by date & available dates)
- JSON-based data storage
- Terminal-based interactive interface

### Technical
- Standard library only (no external dependencies)
- Cross-platform support (Termux/Linux, Windows)
